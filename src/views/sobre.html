Sobre a aplicação

Aplicação para cadastro de clientes de maneira rápida e eficiente.

Detalhes do Projeto

Versão: 1.0

Licença: MIT

Autor: Erica Viana

Descrição: Sistema desenvolvido para ajudar no gerenciamento de dados de clientes. Oferece uma interface simples e fácil de usar para inserção e consulta de informações cadastrais.

Funcionalidades

  • Cadastro de clientes com campos essenciais como nome, endereço e contato.
  • Interface simples e intuitiva.
  • Armazenamento local dos dados (offline).
  • Facilidade de uso em dispositivos móveis e desktops.
